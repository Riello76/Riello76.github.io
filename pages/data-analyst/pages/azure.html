❖ About This Page ❖

During my bootcamp, I completed four hands-on Microsoft Azure projects that showcase my practical skills in cloud data management. These projects include Azure SQL Database for relational data, Azure Cosmos DB for non-relational document-based data, Azure Storage Account for managing unstructured data, and Microsoft Fabric for integrating and analyzing data across multiple sources. Each project strengthened my ability to create, query, and manipulate data in the cloud, build pipelines, and derive actionable business insights.
