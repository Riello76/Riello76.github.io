Northwind Database Create
In this section of the Northwind Database project, the focus is on defining clear and reliable primary keys for the main tables. Primary keys are essential to guarantee that every record can be uniquely identified, making the structure of the database predictable and consistent. By establishing these keys, we strengthen referential integrity, avoid ambiguous or duplicated entries, and create a solid foundation for building relationships among tables such as Customers, Orders, Products, and Suppliers. This step is crucial to ensure accurate joins, faster queries, and a cleaner, more dependable data model.

Northwind Database – Basic Queries
In this section, the focus is on exploring the core SQL operations applied within the Northwind Database. These exercises introduce the essential techniques for retrieving and manipulating data, such as selecting records, filtering rows, sorting results, and counting unique values. You will also work with text patterns, conditional logic, and subqueries, all of which are fundamental tools for real-world database analysis. By practising these queries, you build a strong base for more advanced topics including table relationships, joins, computed fields, and efficient data exploration. Each command is designed to strengthen your understanding of how relational databases operate, ensuring a cleaner workflow and a more structured approach to data handling.

-- Orders and Customers
select * from orders
inner join customers on orders.customerid = customers.customerid;

select customers.customerid, customers.customername, orders.orderid
from orders
inner join customers on orders.customerid = customers.customerid;

-- Combine Orders and Employees
select firstname, lastname, orders.orderid
from employees
inner join orders on employees.employeeid = orders.employeeid
order by orderid;

-- Products and Suppliers
select Productname, suppliername
from products
inner join suppliers on products.supplierid = suppliers.supplierid
where suppliername = "G'Day, Mate";

-- Orders with Employees & Shippers
select firstname, lastname, orders.orderid, orders.orderdate
from employees 
inner join orders on orders.employeeid = employees.employeeid; 

SELECT * FROM orders 
INNER JOIN shippers ON orders.shipperid = shippers.shipperid 
WHERE ShipperName = 'United Package';

-- Products and Categories
SELECT ProductName, CategoryName 
FROM Products 
INNER JOIN Categories ON Products.CategoryID = Categories.CategoryID; 

-- Products and Order Details
select products.productid, products.productname, order_details.quantity 
from products 
inner join order_details on products.productid = order_details.productid;

-- RIGHT and LEFT JOIN examples
SELECT Orders.OrderID, Employees.LastName, Employees.FirstName
FROM Orders
RIGHT JOIN Employees ON Orders.EmployeeID = Employees.EmployeeID
ORDER BY Orders.OrderID;

SELECT Customers.CustomerName, Orders.OrderID
FROM Customers
LEFT JOIN Orders ON Customers.CustomerID = Orders.CustomerID
ORDER BY Customers.CustomerName;

-- Customers without orders
SELECT Customers.CustomerName, Orders.OrderID
FROM Customers
LEFT JOIN Orders ON Customers.CustomerID = Orders.CustomerID
WHERE orderid IS NULL;

-- Aliasing
SELECT c.CustomerName, o.OrderID
FROM Customers as c
LEFT JOIN Orders as o ON c.CustomerID = o.CustomerID
WHERE orderid IS NULL;

-- CROSS JOIN example
SELECT count(*) FROM Customers CROSS JOIN Orders;

-- Aggregates
select avg(price), productname from products;

SELECT COUNT(CustomerID), Country
FROM Customers
GROUP BY Country;

-- Orders per Shipper
select count(orderid) as totalorders, shippername
from shippers as s
inner join orders as o on s.shipperid = o.shipperid
group by shippername;

-- Total sales per product
SELECT p.ProductName, SUM(od.Quantity * p.Price) AS TotalSales
FROM Order_Details as od
JOIN Products as p ON od.ProductID = p.ProductID
GROUP BY p.ProductName
ORDER BY TotalSales DESC;

-- Customers per country
select count(customerid) as noofcustomers, country
from customers
group by country;

-- Total products sold per category
select categoryname, sum(quantity) as totalquantity
from categories as c
join products as p on c.categoryid = p.categoryid
join order_details as od on od.productid = p.productid
group by categoryname;

-- Orders per employee
select firstname, lastname, count(orderid) as totalorders
from employees as e
join orders as o on e.employeeid = o.employeeid
group by firstname, lastname;

select concat_ws(", ", firstname, lastname) as employee,
count(orderid) as totalorders
from employees as e
join orders as o on e.employeeid = o.employeeid
group by firstname, lastname;
