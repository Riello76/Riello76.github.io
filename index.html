Technical Details

For this page, I used Bootstrap for the navigation bar and the carousel, which I customized to adapt to the style of the site. I created the logo and icons with Inkscape, and they are in SVG format. I added scrolling animations using the AOS (Animate on Scroll) library. I created two CSS files: the first one takes care of the resets, the header, and the footer, as they will be the same on all pages. For simplicity and ease of maintenance, I have chosen to have a separate CSS file for this purpose. The second CSS file mainly covers the body of the page, as other pages will be structured similarly. I integrated the CSS for the reset, header, and footer into the main CSS file using @import at the top of the CSS. I used the system-ui font family for better readability and performance. Additionally, I utilized the clamp() function for the font size to ensure it adjusts dynamically between minimum and maximum values for optimal readability across different screen sizes. The hero image was created using AI and then customized with GIMP. I converted and optimized it in WEBP format to ensure faster loading times and better performance.

This portfolio is a work in progress, and I am continually refining and adding new elements to enhance both functionality and aesthetics. You will get a glimpse of my current work, the techniques I use, and my commitment to creating a smooth and engaging user experience. Stay tuned for updates as I continue to build and improve my portfolio.
