About Me

I'm Gaspare Tocci, a junior web developer based in London. I began my journey in 2023, teaching myself tools like GIMP, Inkscape, and Blender, before completing the GetSetUK Web Development Bootcamp.

I’m passionate about clean, readable code, responsive design, and creating user-friendly experiences. I enjoy turning ideas into real projects, learning on the go, and applying new skills practically rather than just theoretically.

I pay attention to details like performance optimization, using modern formats such as SVG and WEBP, and minifying assets for faster load times. I also enjoy photo and video editing, and integrate AI-powered tools into my workflow to boost creativity, improve efficiency, and explore new problem-solving approaches. For me, AI is not a replacement but a supportive tool that helps refine ideas, speed up repetitive tasks, and leave more space for human creativity and decision-making.

I’m currently seeking an apprenticeship or entry-level opportunity where I can contribute, learn, and grow as part of a collaborative, modern team.
My goal is to turn my self-taught foundation into professional experience, and I’m excited to take on new challenges while bringing a creative eye to coding.

Technical Details

This portfolio was built using HTML, CSS, and JavaScript to ensure a lightweight, responsive, and interactive experience. I focused on semantic HTML for accessibility, modular CSS for maintainability, and JavaScript to implement dynamic behaviors like the carousel and scroll animations.

The carousel, buttons, and interactive elements are controlled using plain JavaScript, keeping the code efficient and dependency-free. CSS techniques like flexbox, grid, and clamp() were used for layout and responsive typography. Images and icons are optimized in SVG and WEBP formats for faster load times.

I also implemented smooth scrolling, hover effects, and subtle animations to enhance user experience while prioritizing performance and accessibility. The goal was to create a visually engaging portfolio that works seamlessly across devices.
