Let Me Tell You About Myself!

Welcome to my portfolio! My name is Gaspare Tocci, and I am Italian, currently living in London. I have a great passion for web design and development.

I am a self-taught individual who seizes every opportunity to follow courses that can enhance my knowledge and provide new perspectives. I enjoy working in a team to achieve goals together in the best way possible.

This is my first portfolio, but I have a clear vision: a site that embodies sustainable web design without depriving the visitor of a beautiful experience, incorporating micro animations where appropriate. I strive for a design that is simple to understand and use, while also providing an excellent visual effect.

All my images, logos, and icons are in SVG or WEBP formats. I aim to write code that respects best practices, ensuring everything is minified to increase loading speed by the end of the project.

I use AI as a support tool to speed up writing, and I assist myself with snippets. Although I am still at the beginning of my journey and have much to learn, I am eager for an apprentice opportunity. For my portfolio, I used Bootstrap to help streamline the design and responsiveness of the layout.

Inkscape:
Used for creating and editing vector graphics, ensuring scalable and high-quality visuals. I leveraged its powerful tracing capabilities and drawing tools for designing custom logos, icons, and illustrations.


VS Code:
My preferred code editor for writing clean, efficient code while following best practices. I use extensions such as HTML Preview, Live Preview, Open Browser Preview, Code Spell Checker, and Prettier for real-time updates, spell checking, and automatic code formatting.


Gimp:
Used for image editing and manipulation, enhancing photos, and creating custom graphics. I utilized advanced tools like layers, masks, and filters to achieve professional visual effects.


Blender:
Utilized for 3D modeling and animation, adding dynamic elements to my projects. I created detailed 3D models and smooth animations, integrating them into my web projects for greater interactivity and visual impact.


Figma:
Used for UI design and collaborative prototyping. I created interactive wireframes and mockups, facilitating design idea communication and real-time team collaboration.


Filmora:
A video editing tool used for creating and editing videos with a user-friendly interface. I used its various effects, transitions, and audio editing capabilities to produce polished and engaging video content.


PowerPoint:
Utilized for creating presentations with impactful visuals and clear structure. I designed custom slides, incorporating multimedia elements and animations to enhance the delivery of information and engage the audience effectively.


Git:
Used for version control and collaboration in software development projects. I utilize Git commands for managing code versions, branching, merging, and collaborating with team members to ensure project integrity and efficiency.

OBS:
Used for video recording and live streaming. I leverage OBS's powerful features to capture high-quality video content, configure scenes and sources, and stream live events with customizable overlays and transitions to enhance viewer engagement.

For this page, I used Bootstrap for the navbar, which I customised to fit the style of the site. The logo and icons in the "My Tools" section were created with Inkscape and are in SVG format. I have added scroll animations using the AOS library. I created two CSS files: the first one deals with resets, the header, and the footer, as these will be the same across all pages. For simplicity and easy maintenance, I opted to have a separate CSS file for this purpose. The second CSS file primarily covers the body of the page, as other pages will be structured similarly. I integrated the CSS for resets, header, and footer into the main CSS file using @import at the top of the CSS.

This portfolio is a work in progress, and I am continually refining and adding new elements to enhance both functionality and aesthetics. You will get a glimpse of my current work, the techniques I use, and my commitment to creating a smooth and engaging user experience. Stay tuned for updates as I continue to build and improve my portfolio.
