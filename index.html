Technical Details

This portfolio was built using HTML, CSS, and JavaScript to ensure a lightweight, responsive, and interactive experience. I focused on semantic HTML for accessibility, modular CSS for maintainability, and JavaScript to implement dynamic behaviors like the carousel and scroll animations.

The carousel, buttons, and interactive elements are controlled using plain JavaScript, keeping the code efficient and dependency-free. CSS techniques like flexbox, grid, and clamp() were used for layout and responsive typography. Images and icons are optimized in SVG and WEBP formats for faster load times.

I also implemented smooth scrolling, hover effects, and subtle animations to enhance user experience while prioritizing performance and accessibility. The goal was to create a visually engaging portfolio that works seamlessly across devices.
