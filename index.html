Let Me Tell You About Myself!

About me

I'm Gaspare Tocci, an Italian currently based in London. My journey into web design and development began from scratch in 2023, where I dedicated two months to learning coding and using tools like GIMP, INKSCAPE, and BLENDER. After a brief pause due to some challenges, I resumed my training in April 2024 through the GetSetUK bootcamp, which has significantly advanced my skills.

I thrive as a self-motivated learner constantly seeking growth opportunities. Whether it's through online courses, workshops, or hands-on projects, I am committed to expanding my expertise. I also value collaborative environments where I can contribute to collective goals and learn from others.

This portfolio is my first major project, reflecting my vision for sustainable web design. I aim to create user-friendly interfaces that integrate micro animations to enhance the visitor experience without compromising simplicity or efficiency. My focus is on making designs that are both intuitive and visually appealing.

To ensure optimal performance, I use SVG and WEBP formats for all my images, logos, and icons. My coding practices adhere to industry standards, emphasizing code minification and best practices to enhance loading times and overall project efficiency. However, the Open Graph image is in PNG format for compatibility purposes.

AI tools play a crucial role in accelerating my development process, aiding in coding and writing through snippets. As a newcomer in the field, I am enthusiastic about learning and am actively seeking an apprenticeship. For this portfolio, I utilized Bootstrap to streamline the layout and ensure responsive design.

Technical Details

For this page, I used Bootstrap for the navigation bar and the carousel, which I customized to adapt to the style of the site. I created the logo and icons with Inkscape, and they are in SVG format. I added scrolling animations using the AOS (Animate on Scroll) library. I created two CSS files: the first one takes care of the resets, the header, and the footer, as they will be the same on all pages. For simplicity and ease of maintenance, I have chosen to have a separate CSS file for this purpose. The second CSS file mainly covers the body of the page, as other pages will be structured similarly. I integrated the CSS for the reset, header, and footer into the main CSS file using @import at the top of the CSS. I used the system-ui font family for better readability and performance. Additionally, I utilized the clamp() function for the font size to ensure it adjusts dynamically between minimum and maximum values for optimal readability across different screen sizes. The hero image was created using AI and then customized with GIMP. I converted and optimized it in WEBP format to ensure faster loading times and better performance.

This portfolio is a work in progress, and I am continually refining and adding new elements to enhance both functionality and aesthetics. You will get a glimpse of my current work, the techniques I use, and my commitment to creating a smooth and engaging user experience. Stay tuned for updates as I continue to build and improve my portfolio.
