Specifications

To support the I Want Project identity, this portfolio was engineered from the ground up for maximum performance and zero bloat. Built using Semantic HTML5, Modular CSS3, and JavaScript, the architecture ensures an ultra-lightweight, dependency-free environment.

The core UI utilizes modern layout mechanics like CSS Grid, Flexbox, and fluid mathematical functions like clamp() for flawless responsive typography. To optimize the critical rendering path, all visual assets, including the core Hero Image, are deployed in next-gen WEBP and vector SVG formats, drastically reducing load times.

Dynamic behaviors, custom carousel logic, and layout triggers are driven entirely by raw JavaScript without external frameworks. This architecture balances visual storytelling with optimal Core Web Vitals, ensuring accessibility, speed, and high visibility across all modern devices.
